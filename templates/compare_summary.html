About Summary App

The Summarizer app has been designed using HTML, CSS, JavaScript and Python. The packages used for text summarisation are:

  1. NLTk
  2. Gensim
  3. SpaCy
  4. Sumy
Beautifulsoup is used to scrape content from another website by making use of urllib which fetches the http response from the website.
